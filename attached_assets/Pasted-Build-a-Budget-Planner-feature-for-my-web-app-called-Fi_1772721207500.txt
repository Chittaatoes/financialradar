Build a **Budget Planner feature** for my web app called **Financial Radar**.

The UI should follow the visual references I will upload (screenshots 1–7).
The design style should be modern, minimal, and similar to the Budggt budgeting app but using the Financial Radar green theme.

This feature should allow users to create a monthly budget plan using a step-by-step bottom sheet flow.

---

GENERAL GOAL

Create a budgeting system where users:

1. Enter their expected monthly income
2. Choose a budgeting strategy
3. Allocate money across main financial categories
4. Confirm their budget
5. View their budget dashboard with progress bars showing spending vs limits.

Budget must be **saved per month** and can be **reset each month**.

---

MAIN CATEGORIES

There are four main budget groups:

* Kebutuhan (Needs)
* Keinginan (Wants)
* Tabungan (Savings)
* Investasi (Investment)

Each group will later contain spending categories like food, housing, transport, etc.
