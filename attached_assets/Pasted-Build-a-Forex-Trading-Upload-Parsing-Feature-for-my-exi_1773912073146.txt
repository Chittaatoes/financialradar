Build a Forex Trading Upload & Parsing Feature for my existing web app (React + TypeScript frontend, Node.js + Express backend, PostgreSQL with Drizzle ORM).

## 🎯 Goal

Users can upload a screenshot (MT4/MT5 trading history), and the system will:

1. Parse trading data automatically
2. Normalize symbols (e.g., XAUUSD.sc → XAUUSD)
3. Extract structured trades
4. Prevent duplicate trades from being inserted
5. Allow user to preview/edit before saving

## ⚙️ BACKEND REQUIREMENTS (Node.js + Express)

### 1. Endpoint: POST `/api/forex/parse`

* Accept image (multipart/form-data)
* Use OCR or AI Vision (prefer OpenAI Vision or Tesseract)
* Extract raw text
* Parse into structured trades

### 2. Parsing Rules

From text like:
"XAUUSD.sc buy 2 4833.69 → 4834.32 315.00"

Extract:

* symbol: "XAUUSD"
* type: "BUY"
* lot: 2
* openPrice: 4833.69
* closePrice: 4834.32
* profit: 315.00

### 3. Normalization

* Remove suffix like ".sc"
* Uppercase symbol

```ts
symbol = rawSymbol.replace('.sc', '').toUpperCase()
```

## 🧠 DUPLICATE DETECTION (IMPORTANT)

Before inserting trades, check duplicates:

### Rule:

A trade is considered duplicate if:

* same userId
* same symbol
* same openPrice
* same closePrice
* same type
* AND created within same day

### Implementation:

```ts
const existing = await db.query.trades.findFirst({
  where: and(
    eq(trades.userId, userId),
    eq(trades.symbol, trade.symbol),
    eq(trades.openPrice, trade.openPrice),
    eq(trades.closePrice, trade.closePrice),
    eq(trades.type, trade.type),
    gte(trades.createdAt, startOfDay),
    lte(trades.createdAt, endOfDay)
  )
})
```

If exists:

* SKIP insert
* mark as duplicate in response
