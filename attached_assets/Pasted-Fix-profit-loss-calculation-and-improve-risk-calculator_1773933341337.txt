# ✅ STEP 1: CALCULATE PROFIT DIRECTION

Rule:

BUY (LONG):

* profit if closePrice > openPrice
* loss if closePrice < openPrice

SELL (SHORT):

* profit if closePrice < openPrice
* loss if closePrice > openPrice

---

## IMPLEMENTATION

```ts
function calculateProfit({
  type,
  openPrice,
  closePrice,
  lot
}) {
  const diff = closePrice - openPrice

  let raw = 0

  if (type === 'BUY') {
    raw = diff
  } else {
    raw = -diff
  }

  // XAUUSD approx: $10 per 1 lot per 1.0 move
  const multiplier = 10

  const profit = raw * lot * multiplier

  return Number(profit.toFixed(2))
}
```

function normalizeLot(lot, accountType) {
  if (accountType === 'CENT') {
    return lot * 0.01
  }
  return lot
}
