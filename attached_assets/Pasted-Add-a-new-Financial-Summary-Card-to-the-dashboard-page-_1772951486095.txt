Add a new **Financial Summary Card** to the dashboard page of the finance tracker app.

The card should be placed **below the greeting ("Hi, Guest") and above the Total Assets card**.

The purpose of this card is to give users a quick overview of their financial status for the current month.

### Card Title

Today's Budget

### The card must contain these sections:

1. **Budget Aman Hari Ini (highlighted number)**

* This should be the largest number in the card.
* Formula:
  remainingMonthlyBudget / remainingDaysInMonth

2. **Monthly Income**
   Total income recorded for the current month.

3. **Monthly Expenses**
   Total expenses recorded for the current month.

4. **Remaining Budget This Month**
   Formula:
   monthlyIncome - monthlyExpenses

5. **Progress Bar**
   Show percentage of expenses compared to income.

Formula:
(monthlyExpenses / monthlyIncome) * 100

### Layout

Top section:

* Title: "Today's Budget"
* Big highlighted number: Budget Aman Hari Ini

Middle section:
Two columns:

* Pemasukan Bulan Ini
* Pengeluaran Bulan Ini

Bottom section:

* Remaining Budget This Month
* Progress bar indicating how much of the budget has been used.

### UI Style

* Use the same **rounded card design** as other dashboard cards.
* Apply the existing **green gradient theme** used in the app.
* Mobile-first responsive design.
* Clean fintech style UI.

### Extra UX detail

Add a small status indicator based on spending pace:

Green → "On Track" (expenses ≤ expected spending)
Yellow → "Be Careful" (expenses slightly above expected)
Red → "Overspending"

Expected spending formula:
(monthlyIncome / daysInMonth) * daysPassed

### Data sources

Use existing transaction data already stored in the app to calculate:

* monthlyIncome
* monthlyExpenses

### Goal

This card should give users an instant understanding of:

* how much they can safely spend today
* how much they already spent this month
* how much budget remains.
