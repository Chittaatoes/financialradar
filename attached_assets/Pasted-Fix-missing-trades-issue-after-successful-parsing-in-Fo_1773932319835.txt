# ✅ STEP 1: FIX DUPLICATE DETECTION (CRITICAL)

Current duplicate logic is too aggressive.

UPDATE rule:

A trade is duplicate ONLY IF:

* same userId
* same symbol
* same openPrice
* same closePrice
* same type
* AND timestamp is within ±1 minute

---

### Add timestamp tolerance:

```ts
const isDuplicate = await db.query.trades.findFirst({
  where: and(
    eq(trades.userId, userId),
    eq(trades.symbol, trade.symbol),
    eq(trades.openPrice, trade.openPrice),
    eq(trades.closePrice, trade.closePrice),
    eq(trades.type, trade.type),
    gte(trades.createdAt, new Date(trade.createdAt - 60000)),
    lte(trades.createdAt, new Date(trade.createdAt + 60000))
  )
})
```
