# ⚙️ BACKEND

## 1. POST `/api/forex/parse`

* Accept image
* Use OCR or AI Vision
* Extract text
* Parse trades

---

## 2. Parsing Logic

From:
"XAUUSD.sc buy 2 4833.69 → 4834.32 315.00"

Extract:

* symbol: XAUUSD (remove .sc)
* type: BUY
* lot: 2
* openPrice
* closePrice
* profit

---

## 3. Symbol Normalization

```ts
symbol = rawSymbol.replace('.sc', '').toUpperCase()
```

---

## 4. DUPLICATE DETECTION (CRITICAL)

A trade is duplicate if:

* same userId
* same symbol
* same openPrice
* same closePrice
* same type
* same day

If duplicate:

* DO NOT insert
* return flag: duplicate = true

---

## 5. POST `/api/forex/save`

* Insert only non-duplicate trades
* Return:
  {
  inserted: number,
  duplicates: number
  }

# 🧠 PSYCHOLOGY SYSTEM (IMPORTANT FEATURE)

## 1. Daily Loss Limit

User setting:

* maxLossPercent = 1%

Logic:
If totalLossToday >= balance * 0.01:

* block new trade input
* show warning:
  "Kamu sudah mencapai batas kerugian harian"

---

## 2. Profit Target Reminder

If profit >= target (ex: 2%):
Show popup:
"Profit sudah cukup. Mau lanjut atau istirahat?"

Buttons:

* Lanjut (danger)
* Istirahat (recommended)

---

## 3. Overtrading Detection

If trades > 10/day:
Show warning:
"Overtrading terdeteksi"

---

## 4. Revenge Trading Detection

* last trade = loss

"Tenang dulu. Jangan balas dendam market."
