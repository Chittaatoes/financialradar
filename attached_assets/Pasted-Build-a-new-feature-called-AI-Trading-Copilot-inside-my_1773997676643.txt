Build a new feature called **"AI Trading Copilot"** inside my existing Forex page (React + TypeScript frontend, Node.js backend deployed on Render).

This feature integrates with the TradingAgents framework (https://github.com/TauricResearch/TradingAgents) to generate AI-based trading decisions (LONG / SHORT / NO TRADE) for forex pairs.

---

## 🎯 MAIN GOAL

Create an AI-powered assistant that:

* Analyzes a selected forex pair
* Uses multi-agent reasoning (technical, news, sentiment)
* Produces a structured trading decision
* Explains the reasoning (Explainable AI)
* Integrates with existing trading journal & risk system

---

### 1. AI Analysis Panel (Frontend Component)

Create a React component:

`AIForexCopilot.tsx`

* Pair selector (EUR/USD, GBP/USD, etc)

interface AIForexSignal {

## ⚙️ BACKEND IMPLEMENTATION

### 1. Create new endpoint:

POST `/api/forex/ai-analyze`

### 2. Backend Architecture

Node.js (Render) acts as proxy → Python TradingAgents service

Flow:
Frontend → Node API → Python AI → Node → Frontend

---

### 3. Python Service (TradingAgents)

Create:

`/python-ai/forex_ai.py`

Use TradingAgentsGraph.

Adapt stock logic → forex pair:

Example:

```python
pair = "EURUSD"
```

Return structured response:

```json
{
  "direction": "LONG",
  "entry": 1.0870,
  "stop_loss": 1.0840,
  "take_profit": 1.0950,
  "confidence": 78,
  "risk_level": "medium",
  "style": "swing",
  "agents": {
    "technical": "bullish",
    "news": "neutral",
    "sentiment": "bullish",
    "fundamentals": "bullish"
  },
  "debate": {
    "bull": "...",
    "bear": "...",
    "judge": "..."
  },
  "reasoning": "Detailed explanation..."
}
```

---

### 4. Node Route Example

```ts
app.post('/api/forex/ai-analyze', async (req, res) => {
  const { pair } = req.body;

  const response = await fetch('http://python-service/analyze', {
    method: 'POST',
    body: JSON.stringify({ pair })
  });

  const data = await response.json();

  res.json(data);
});
```
