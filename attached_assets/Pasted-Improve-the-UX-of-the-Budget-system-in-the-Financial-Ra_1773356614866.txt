Improve the UX of the Budget system in the Financial Radar app so the budget always feels active and guided for the user.

IMPORTANT:
Do NOT modify any existing business logic, calculations, backend APIs, or database schema.

The goal is to improve user experience only.

Budget calculations and logic must remain exactly the same.

---

1. DETECT BUDGET PERIOD STATUS

---

Use the existing budget summary data that already provides:

periodStart
periodEnd

Compute on the frontend:

daysRemaining = periodEnd - today

Create three states:

ACTIVE
ENDING_SOON
EXPIRED

Rules:

ACTIVE
daysRemaining > 3

ENDING_SOON
daysRemaining ≤ 3 AND daysRemaining ≥ 1

EXPIRED
today > periodEnd

---

2. ADD BUDGET STATUS INDICATOR

---

In the dashboard "Today's Budget" card, show a status indicator.

ACTIVE

Display current indicator:

On Track

ENDING_SOON

Display:

Budget period ending soon

EXPIRED

Display:

Budget expired — set up your new budget

---

3. SHOW BUDGET REMINDER BANNER

---

When daysRemaining ≤ 2 show a soft reminder banner above the budget card.

Example message:

"Your budget period ends in 2 days. Prepare your next budget."

Include button:

Review Budget

This button navigates to the Budget page.

---

4. BUDGET EXPIRED PROMPT

---

When the user opens the app and the budget is expired:

Show a bottom sheet modal.

Title:

New budget period started

Message:

Your previous budget period has ended.
Would you like to reuse your previous budget setup?

Buttons:

Use Previous Setup
Edit Budget
Skip for Now

---

5. REUSE PREVIOUS BUDGET SETUP

---

If the user selects "Use Previous Setup":

Automatically navigate to the Budget page with the previous configuration prefilled.

Do NOT create or modify backend logic.

Simply load the existing values into the UI form.

---

6. ADD DASHBOARD BADGE

---

If the budget is expired:

Add a small badge on the dashboard budget card.

Example:

⚠ Budget needs setup

Clicking the badge opens the Budget page.

---

7. ADD BUDGET REMINDER NOTIFICATION

---

Add a daily check that runs when the dashboard loads.

If:

daysRemaining == 2

Show toast:

"Your budget period will end soon."

If:

budgetExpired

Show toast:

"Your budget period has ended. Set up your new budget."

---

8. KEEP CURRENT SAFE DAILY BUDGET

---

Do NOT modify any Safe Daily Budget calculations.

Only adjust UI messaging.

---

9. FINAL RESULT

---

After this update:

The budget system always feels active.

Users will always know when their budget period is:

active
ending soon
expired

Users will be guided to create a new budget cycle without breaking any existing logic.

The backend and budget calculations must remain unchanged.
